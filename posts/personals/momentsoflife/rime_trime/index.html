rime-ice 雾凇拼音 + trime 同文输入法

怎么在手机上使用这个输入法？比较无脑省力那种…

应该说明：这篇文章基本是我自己的备忘录，我自己已经使用这些配置多年。 如果你是第一次接触 rime/Trime，直接查看他们官网或者本文的参考链接，否则如果按着我的说法进行操作，大概率你会搞得自己火冒三丈。

我的操作：

1.1 手机上下载 F-Droid，在 F-Droid 搜索并下载 Trime 输入法 (中文名：同文输入法1)

1.2 手机应用授权等其他操作

2.1 电脑端下载仓库

cd rime_hub
git clone https://github.com/iDvel/rime-ice.git

2.2 将手机接入电脑，将仓库复制到手机目录路径（通常是与Download,Documents同级的路径），将 其名称修改为 rime (rime-ice也可以，但要做出相应的适配和设置)

3.1 手机端打开 Trime 输入法，点击右上角部署

3.2 可能会报错弹出日志窗口，但不一定影响使用；否则，去官网等地方寻找解决方案2

4.1 如果也在电脑端使用，可以借助 Plum 进行安装和更新：

git clone --depth=1 https://github.com/rime/plum
cd plum
bash rime-install iDvel/rime-ice:others/recipes/full

#全量更新：
bash rime-install iDvel/rime-ice:others/recipes/full

#局部更新：
bash rime-install iDvel/rime-ice:others/recipes/all_dicts

5 多端同步：电脑端更新，复制到手机等其他终端。

  2. Rime 配置：雾凇拼音 https://dvel.me/posts/rime-ice/ ↩︎
